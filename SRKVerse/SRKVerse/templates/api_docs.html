SRKVerse API Documentation

Access Shah Rukh Khan's data programmatically.

Endpoints

  • GET /api/movies/ - List all movies
  • GET /api/movies/{id}/ - Get movie details
  • GET /api/bio/ - Get biography

Example Request

curl https://api.srkverse.com/movies/1/

Sample Response

{
    "id": 1,
    "title": "Dilwale Dulhania Le Jayenge",
    "year": 1995,
    "plot": "A romantic journey of two young hearts..."
}

Note: API key required for access (contact support).
